# WebCrawler Infosys Project

## 📌 Project Overview
This project is a **Mini Search Engine** built using Python.  
It crawls web pages, stores data, and allows users to search information.

---

## 🚀 Features
- Web crawling using Python
- Data indexing
- Search functionality
- Fast result retrieval

---

## 🛠️ Technologies Used
- Python
- Flask (if used)
- HTML, CSS
- JSON

---

## 📂 Project Structure
- crawler.py → Crawls web pages
- main.py → Main execution file
- indexer.py → Creates index
- worker.py → Background processing
- documents.json → Stored data
